# coursehose

## 1. Installation  
(might be slightly off oops)

Go into the correct directory: `cd vue-app`  

Install packages: `npm install`  

Install Vue CLI: `npm i -g @vue/cli @vue/cli-service-global`  
`yarn global add @vue/cli @vue/cli-service-global`

Start the server:
`yarn serve`
