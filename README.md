# coursehose

## 1. Installation  
(might be slightly off oops)
`cd vue-app`  
`npm install` 
`npm i -g @vue/cli @vue/cli-service-global`  
`yarn global add @vue/cli @vue/cli-service-global`
`yarn serve`
